## MVP

- Add:
  - At the beginning;
  - At the End;
  - Random;
- Remove:
  - All tasks (with a confirmation modal);
  - First;
  - Last;
  - Random;
  - Duplicates;
  - Odd/Even;
- Sort:
  - Random;
  - Sort Alphabetically (A-Z/Z-A);
  - Reverse;
- Save/load state from localstorage;
- Individually:
  - Remove
  - Mark as complete
  - Edit;

### Nice to Have
- Search bar to filter the task with the current input;
- Filters (complete, incomplete);
